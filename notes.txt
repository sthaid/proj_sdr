Setup the SDR
-------------

start.nesdr.com
https://www.nooelec.com/store/downloads/dl/file/id/72/product/0/nesdr_installation_manual_for_ubuntu.pdf

Steps:
- vi /etc/modprobe.d/blacklist-dvb.conf
  blacklist dvb_usb_rtl28xxu
  reboot
- apt install rtl-sdr
- rtl_test
- Ubuntu Software -> install gqrx  (takes some time)
